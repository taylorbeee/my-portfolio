Get to know me!

Hey there! I'm Taylor Behrens and I'm a Front End Web Developer who loves to bring digital projects to life. I have always loved solving puzzles and chasing the dopamine rush of finding the answer to a tough problem.

I began my journey in development in college with a few intro classes to HTML, CSS and Java. Soon enough I found myself at a crossroads - do I continue on this route to a computer science profession? Or take the far more stable career path of an actor/writer? Hard-cut to me now, where I have rediscovered my love for coding and problem-solving. My entertainment background is now my secret weapon as it has trained me to think out of the box and find new, creative ways to resolve issues, communicate ideas and work with expansive, diverse teams.

When I'm not coding you can find me, listening to film podcasts, reading sci-fi, or bringing up the rear at my local running club.

Sticky Note UI

An interactive app that will let users create “sticky notes” on their computer to keep track of to-do items. This project utilizes stateful components in React. By passing attributes via props and using the .map method, the user can add, delete and edit their sticky notes, and using lifecycle methods, the program remembers each note, so that if the user leaves, when they return all the previous notes are still there.

SKILLS: React, JSX, ES6, Responsive Design, Data Flow, DOM Manipulation, Local Storage
